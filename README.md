# Visualizador de Garantías

App de Streamlit para visualizar y gestionar garantías, con subida y descarga de archivos Excel por comitente.

## Requisitos

- Python 3.8+
- Streamlit
- Pandas
- Openpyxl
- XlsxWriter

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
